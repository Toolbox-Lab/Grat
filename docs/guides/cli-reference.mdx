The CLI is the primary interface for Grat. Every feature is accessible from the terminal with human-readable colored output.

## Essential Commands

### `grat decode <tx-hash>`
The most common command. Fetches and decodes a transaction failure.
- `--network`: `mainnet`, `testnet` (default), or a custom RPC URL.
- `--output json`: For machine-readable results.

### `grat trace <tx-hash>`
Replays the transaction and outputs a hierarchical execution timeline.

### `grat replay <tx-hash> --interactive`
Launches the TUI debugger for step-through investigation.

### `grat profile <tx-hash>`
Identifies budget hotspots and expensive operations.

### `grat serve`
Starts a WebSocket server for streaming trace updates to the web interface.

### `grat db update`
Downloads the latest version of the Error Taxonomy Database.
